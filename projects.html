Dec 2025 · UC Irvine

Thermochemical Nonequilibrium Shock Modeling

Solo project · MATLAB / Python

Built a 1-D nitrogen shock-layer model using Rankine–Hugoniot jump conditions and a two-temperature N₂/N formulation to capture thermochemical nonequilibrium behind a strong shock.

  • Coupled ODE solver marching species, velocity, and translational + vibrational temperatures
  • Captured post-shock relaxation until the flow returns to equilibrium
  • Post-processed results into engineering plots and energy-mode population distributions
